Return Policy


If you should need to return and item all qualified returns must follow RMA return procedures, the following is return qualifications and guidelines which must be followed


  1. All returns or other matters must be made within 30 days of invoice days.
  2. You must get an RMA before returning an item (NOT ALL ITEMS ARE RETURNABLE)
  3. All qualified returns will incur a 25 % re-stock fee, unless a mistake was made by Spa Parts Depot or the part ordered was incorrect and the intent is to order the correct part.
  4. All authorized returns must be shipped prepaid and carefully boxed.
  5. You may use any carrier, but we highly recommend insuring your return and get a tracking #.
  6. Shipping charges will only be refunded in the event an error of Spa Parts Depot

Non-returnable items:


  • Used items
  • Special orders
  • Spa & Bath Control Systems
  • Spa Cover (Please double check your measurements to make sure you have the correct we will have it made per your measurements)
  • Transformers
  • Sensors
  • Topside Control
  • Circuit boards which have been installed can NOT be returned, if the board ordered was incorrect and the intent is to order the correct one it can be exchanged for the correct board with out restock fee.
  • Mechanical parts such as Time Clocks, Contactors, Relays, Thermostats, Hi-Limits, etc.

Clearance Sale Item have a 90 Day Warranty
